## Short Description (132 chars max)

```
Detects and validates Quranic verses on any webpage. Highlights authentic verses in green, flags fabricated Arabic from LLM output.
```

## Detailed Description
```
Mizan detects Quranic verses on any webpage you visit, validates them instantly, and highlights authentic text while flagging fabricated Arabic.

Built for the AI era: large language models confidently invent Quranic verses that look authentic but don't exist in scripture. Mizan catches those hallucinations by checking detected Arabic against a pre-computed index of all 6,236 Quranic verses using 6-stage Arabic normalization.

What it does:
• Scans any webpage for Arabic text that appears to be Quranic
• Validates each segment against the complete Uthmani Quran corpus
• Highlights authentic verses in green, flagged text in red
• Shows verse reference (surah:ayah) in a tooltip on hover
• Works on all websites, including AI chatbot interfaces, social media, and blogs

Works entirely offline after first load - the full Quran data is bundled in the extension. No external API calls, no data collection, no tracking.

Open source: github.com/azrirefik/mizan-checker
```
